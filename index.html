1. JavaScript คืออะไร

ความหมาย

JavaScript คือ ภาษาการเขียนโปรแกรมที่ใช้สร้างเว็บไซต์ให้สามารถโต้ตอบกับผู้ใช้งาน เช่น ปุ่มกด เมนู ตรวจสอบข้อมูล และแสดงข้อความต่าง ๆ

ประวัติความเป็นมา

  • สร้างโดย Brendan Eich
  • พัฒนาในปี ค.ศ.1995
  • พัฒนาที่บริษัท Netscape
  • ปัจจุบันใช้มาตรฐาน ECMAScript

ความสำคัญ

  • ทำให้เว็บไซต์มีความโต้ตอบ
  • ทำงานร่วมกับ HTML และ CSS
  • ใช้สร้าง Web Application
  • ได้รับความนิยมทั่วโลก

2. การทำงานของ JavaScript

Client-side

JavaScript ทำงานภายใน Web Browser ของผู้ใช้งาน เช่น Chrome, Edge และ Firefox

การทำงานร่วมกับ HTML และ CSS

  • HTML สร้างโครงสร้างเว็บไซต์
  • CSS ตกแต่งเว็บไซต์
  • JavaScript เพิ่มการทำงาน

การประมวลผลผ่าน Browser

ผู้ใช้เปิดเว็บไซต์
↓
Browser โหลด HTML
↓
Browser โหลด CSS
↓
Browser ประมวลผล JavaScript
